# 用戶商店端：瀏覽與購買

---

## 導覽列

```
商品列 | 尺寸指南 | 客製化商品 | 🔍 搜尋 | 購物車（件數）| 登入 / 會員
```

**搜尋功能**
- 關鍵字搜尋：`products.title`、`products.description`、`tags.name`，使用 PostgreSQL `ILIKE` 查詢
- 搜尋結果頁支援同商品列表篩選條件

## 商品列表

### 三種分類入口

| 入口 | 說明 |
|------|------|
| 難易度分類 | 入門 / 初級 / 中級 / 進階 四個子分類，點入後顯示該難易度的所有商品 |
| 標籤主題分類 | 顯示所有標籤，點入標籤後顯示該主題的所有商品 |
| 細緻度分類 | 粗糙 / 標準 / 細緻 / 高級，點入後顯示含該細緻度的商品 |

**分頁**：每頁顯示 24 筆商品卡片，下方顯示頁碼導航。

### 列表頁篩選（進入分類後可再篩選）

- 難易度（多選）
- 細緻度（多選）
- 畫布尺寸（多選）
- 標籤（多選）
- 排序：最新上架 / 熱門 / 價格低到高 / 價格高到低

### 商品卡片

- 封面圖
- 商品名稱
- 難易度標籤
- 價格區間（NT$XXX ~ NT$XXX，最低至最高變體價格）
- 庫存狀態（預購中標示）

## 商品詳情頁

### 基本資訊

- 圖片輪播：封面圖 + `product_images` 一起顯示，可左右切換；選擇規格後額外切換顯示該變體的 `filled_template.png`
- 商品名稱
- 商品描述

### 同系列商品

- 若商品屬於某系列，顯示「同系列商品」橫列
- 點選可切換至同系列其他商品
- **排序**：依 `products.series_order` ASC；相同時依 created_at ASC

### 規格選擇

客戶依序選擇：

1. **畫布尺寸**（按鈕選項，只顯示有上架的尺寸）
2. **難易度**（按鈕選項，依選定尺寸過濾可用選項）
3. **細緻度**（按鈕選項，依前兩項過濾可用選項）

- 無對應變體的組合顯示為不可選（灰色）
- 選完後顯示：該變體售價、預覽圖、庫存狀態（現貨 / 預購中）

規格括號說明自動帶入，例如：
> 30×40 cm · 入門 · 18色

### 加入購物車

- 可選購買數量（預設 1）
- 若庫存不足，顯示「現貨 X 件，超出數量為預購」提示
- 點「加入購物車」：
  - 未登入 → 前端將 `variant_id` + 數量暫存於 `sessionStorage`（key：`pending_cart_item`），跳轉登入頁（帶 `redirect` 參數）；登入後重導回商品頁，前端自動執行加入購物車並清除 `sessionStorage`，顯示成功通知（Toast）
  - 已登入 → 直接加入，顯示成功通知（Toast）

### 商品說明折疊區塊

商品頁底部顯示可展開的說明區塊，內容全站共用，由管理員後台統一編輯（存於 `system_settings`）：

| 區塊 | key | 說明 |
|------|-----|------|
| 畫具內容物 | product_info_tools | 套件內含的筆刷、顏料罐、畫布等說明 |
| 畫布材質 | product_info_material | 畫布材質、框架規格說明 |
| 使用建議 | product_info_tips | 適合對象、難易度說明、建議使用環境 |
| 注意事項 | product_info_notes | 保存方式、清潔、退換貨提醒等 |

- 預設全部收起，點擊標題展開
- 所有商品頁共用同一份內容，不需逐商品維護

---

### 客製此規格

- 商品詳情頁底部有「想要不同規格？提交客製申請」按鈕
- 點選後展開客製規格表單（`custom_spec` 類型）：
  - 參考商品（自動帶入目前商品）
  - 畫布尺寸（自動帶入目前選擇的尺寸，若尚未選擇則留空，可修改）
  - 難易度（自動帶入目前選擇的難易度，若尚未選擇則留空，可修改；含「讓管理員建議」選項）
  - 細緻度（自動帶入目前選擇的細緻度，若尚未選擇則留空，可修改；含「讓管理員建議」選項）
  - 備注（選填）
- 提交後建立 `custom_request`，導向「客製申請」頁面追蹤進度
- 管理員報價 → 客戶在報價確認頁確認 → 系統建立獨立訂單進入付款流程
- 客製申請訂單與購物車訂單各自獨立，不合併結帳

## 購物車

訪客只能瀏覽商品，無法使用購物車。點「加入購物車」時跳轉登入頁，登入後重導回原本頁面，商品自動加入購物車。

### 購物車頁面

| 區塊 | 說明 |
|------|------|
| 商品列表 | 商品名稱、規格、單價、數量調整、小計、刪除 |
| 庫存拆單提示 | 若某項目數量超過庫存，顯示「現貨 X + 預購 Y」 |
| 折扣券 | 輸入框（手動輸入 public_code）或從持有券列表選擇 |
| auto_checkout 促銷 | 符合條件時自動顯示已套用的促銷折扣 |
| 免運進度條 | 顯示距免運門檻（NT$800）還差多少，或已達 3 件免運（「件」定義：購物車內所有商品數量加總 ≥ 3）|
| 取貨方式 | 宅配 / 7-11 / 全家，含運費顯示 |
| 超大尺寸警告 | 購物車內含任一邊 >40cm 的商品（如 40×60、60×40、50×50 等）且取貨方式選超商時顯示警告，建議改選宅配 |
| 金額明細 | 小計、折扣、運費、總計 |
| 前往結帳 | 需登入 |

## 結帳頁

1. 確認訂單明細
2. 填寫 / 選擇收件資訊（從已儲存的 shipping_profiles 選，或新填）
   - 超商取件：透過第三方選店 API（7-11 / 全家各自提供門市地圖）選擇門市
   - **行動裝置相容性**：選店地圖預設以 popup 視窗開啟，但行動裝置瀏覽器可能封鎖。前端需偵測 popup 是否被封鎖，若封鎖則降級為**頁面內 iframe** 或**直接跳轉至選店頁面**，選完後導回結帳頁並帶回門市資訊（store_id、store_name）。
3. 確認金額（含折扣、運費）
4. 含預購項目時，選擇出貨偏好（合併出貨 / 分開出貨）
5. 送出訂單 → 系統顯示付款帳號（從 system_settings 讀取）與 24 小時付款期限
6. 同時發送訂單確認 email

## 資料庫相關

**商品系列表（product_series）**

| 欄位 | 說明 |
|------|------|
| id | 主鍵 |
| name | 系列名稱 |
| description | 系列描述（選填）|
| created_at | 建立時間 |

> products 表新增 `series_id`（FK → product_series.id，nullable）與 `series_order`（系列內排序）。

**刪除規則**：系列下仍有商品時禁止刪除，後端回傳錯誤提示管理員先將所有商品移出系列。
